Общие сведения.
Данный программный продукт предназначен для автоматизации учета затрат на основе кассовых чеков.

Возможности:
- хранение инфорации по затратам;
-	полуавтоматическое распознавание чеков;
- вывод статистики по тратам за период в виде графиков.

Платоформы: MS Windows, Linux.
Зависимости: 
1. Qt(4.8) 
2. qwt(6.0.1) 
3. tesseract(3.0.2) 
4. opencv(2.4.3)
5. leptonica (1.69)

Процесс установки.
Windows


Linux
1. qmake
2. make

Краткое руководство пользователя.

В главном окне программы отображен список последних добавленных чеков. Здесь можно задать период отображения добавленных чеков.
	Для того, чтобы добавить новый чек в систему, нужно нажать на кнопку "Добавить чек". Откроется окно выбора файла. Выберите файл со сканированным изображеним чека и нажмите кнопку "Open". Откроется окно для добавления нового чека в систему. Слева находится информация по чеку: наименования товаров, стоимость, а также категория. Внизу дата и общая стоимость. Справа - изображение чека. При необходимости откорректируйте данные и нажмите кнопку "OK". Добавленный чек появится в списке. Для отмены операции нажмите кнопку "Отмена". 
	Для просмотра статистики по затратам нажмите кнопку "Статистика". Откроется окно просмотра статистики. Просмотр информации можно осуществлять по категориям и товарам. Задайте период и нажмите кнопку "Показать". В области просмотра диаграммы появится график затрат.
